## 安装与环境配置

### ubuntu下安装python3, matplotlib, numpy

```sh
$ sudo apt-get install python3
$ sudo apt-get install python3-matplotlib
$ sudo apt-get install python3-numpy
$ sudo apt-get install python3-scipy
```
